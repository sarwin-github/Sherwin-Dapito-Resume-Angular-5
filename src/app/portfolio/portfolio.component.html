Projects and Website

List of Projects and Website I created, starting from January 2017
Ease Travel Service

A one-stop shop for travel in Africa, to bring together in a single application the best service providers in the field of travel, tourism and leisure: Air and land transport, Accommodation, Tourist and cultural activities, Insurance travel, Tourist and Economic Guide. Created using angular.

Check Link: https://ease-ts.com



The Brand Tool

A Social Media and Domain management web application that allow a user to search for brand name or username availability across multiple social media platforms and domain. If the target domain or social media is available a user also has options to register to it with just one click. This app was created using the MEAN stack.

Check Link: https://thebrandtool.com/



Minerva IO

Implemented Frontend design based on a wireframe given. It is a frontend of a data mining app (https://mnrva.io). The app was created using Angular 6, TypeScript, and RxJS.

Check Link: https://thebrandtool.com/



Restaurant Web App

A Sample Restaurant Web App for Kamayan sa Palaisdaan Restaurant, Hotel and Resort where you can check their menu, hotel prices, services and other cool stuff. Created using Angular 6 and Typescript.

Check Link: https://kamayan-sa-palaisdaan-ph.herokuapp.com



Fitness Market App

A fitness market web application created using Node JS, Express, MongoDB, Embedded JavaScript (EJS), Bootstrap 4, Wordpress API, Session Based authentication using passport, and JQuery. Fitness market app offer services such as promoting fitness center and fitness professional or trainer including classes booking and fitness centers membership.

Check Link: https://entrenami.herokuapp.com



Dentistry Web App with Booking Calendar

A sample dentistry website app with booking calendar using FullCalendar api, Jquery, Node JS, MongoDB, Bootstrap 4 and Embedded JavaScript (EJS). This dentistry website app is a sample booking app using full calendar (authentication hasn't been implemented yet for testing).

Check Link: https://dentistry-app.herokuapp.com



Photography Website

A sample photography blog website created using Node JS, Embedded JavaScript (EJS), Bootstrap 4 and JQuery. This photography website showcases the skill and portfolio of the photographer.

Check Link: https://zaspix.herokuapp.com



Weather and Timezone App

A Weather and Timezone application where you can check the current weather (OpenweathermapAPI) and timezone(timezoneapi.io) based on country and city. It also uses Google map API for geolocation. It was created using Angular 6 and typescript.

Check Link: https://weather-ng.herokuapp.com



Company Website

A sample professional company website created using Node JS, Embedded JavaScript (EJS), Bootstrap 4 and JQuery.

Check Link: https://company-web-solutions.herokuapp.com



Public Chat Application

A sample chat application created using Node JS, Embedded JavaScript (EJS), Socket.IO, MongoDB, Bootstrap 3 and JQuery.

Check Link: https://public-chat-app.herokuapp.com/
